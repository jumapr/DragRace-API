Welcome to the RuPaul's Drag Race API!

This API returns data about contestants and episodes on RuPaul's Drag Race Seasons 1-13. All Stars and other franchises are not yet included. Contestant data includes contestants' name, age, hometown, outcome in the competition, and season. Episode data includes air date, main challenge description, number (within the season), episode title, and episode winner. Main challenge and winner are null if the episode did not include a main challenge (e.g. reunion episodes).

/contestants/all

This API returns the data for all contestants.

Example (click for JSON output)

http://127.0.0.1:5000/all/contestants

/contestants/search

The search route allows searching for data on contestants matching one or more of the available parameters.

API Parameters

Optional: name

Name of a contestant. Partial matches are included.

Optional: outcome

The outcome, or place of a contestant. Valid options are Winner, Runner-Up, Disqualified and places 3rd through 15th. Both ordinal numbers (e.g. 4th) and cardinal numbers (e.g. 4) are accepted.

Optional: season

Season of the television show. Valid options are 1 through 13.

Optional: minage

Minimum age of contestants to search for. The minimum age allowed on the U.S. show is 21.

Optional: maxage

Maximum age of contestants to search for. The oldest contestant on the show thus far was Charlie Hides at age 52.

Examples (click for JSON output)

http://127.0.0.1:5000/search/contestants?name=Monsoon

http://127.0.0.1:5000/search/contestants?season=4

http://127.0.0.1:5000/search/contestants?outcome=5th

/episodes/all

This API returns the data for all episodes.

Example (click for JSON output)

http://127.0.0.1:5000/all/episodes

/epsiodes/search

The search route allows searching for data on episodes matching one or more of the available parameters.

API Parameters

Optional: season

Season of the television show. Valid options are 1 through 13.

Optional: after

Minimum air date of episodes to include. Date must be in YYYY-MM-DD format.

Optional: before

Maximum air date of episodes to include. Date must be in YYYY-MM-DD format.

Examples (click for JSON output)

http://127.0.0.1:5000/search/episodes?season=4

http://127.0.0.1:5000/search/episodes?after=2014-04-07&before=2015-03-02
